# Wiki bundle data (local only)

The full RS3 wiki index (`wiki.rs3wi`, ~460 MB) is **not** in git — GitHub rejects files over 100 MB.

The **GitHub Pages tracker** does not need these files. They are only for the Python guide CLI (`tools/guide_agent.py`).

Copy your bundle data here locally if you use the CLI. See [`../README.md`](../README.md).
