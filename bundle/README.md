# RuneScape Wiki Knowledge Bundle

Portable package for AI agents. **Copy this entire folder** into any project — no indexer code required.

## Contents

| File | Purpose |
|------|---------|
| [`RULES.md`](RULES.md) | **Read first.** Reasoning rules for any wiki goal |
| [`MANIFEST.json`](MANIFEST.example.json) | Build metadata and file paths (generated locally; see example) |
| `data/wiki.rs3wi` | Full wiki page index (generated by build) |
| `data/wiki.rs3wg` | Correlation graph with extracted intel (generated by build) |
| `data/wiki.rs3nav` | Facet index for discovery without scanning all pages |

## Quick start (AI agent)

1. Read `RULES.md` completely
2. Load `data/wiki.rs3wi`, `data/wiki.rs3wg`, and `data/wiki.rs3nav` (format specs are in each file header)
3. Check `wiki.rs3nav` `schema` for discovery paths, then apply RULES §3 for the user's goal
4. Cite wiki page titles and excerpts in your output

## Validate integrity (local)

From `indexer/`:

```bash
python3 wiki.py integrity          # structure + sample record checks
python3 wiki.py integrity --full   # decompress every index record (slow)
python3 wiki.py -v integrity       # include warnings and size notes
```

Does not contact the wiki — verifies files are present, indexes align, graph edges are consistent, and nav pageids join to the index.

## Example workflow

**User asks:** "How do I get a Bandos godsword from a level 3 account?"

1. Resolve `Bandos godsword` in the index
2. Traverse prerequisites backward on the graph (`requires_material`, `drops`, `requires_skill`, `requires_quest`)
3. Compare skill/quest gates to account state
4. Apply RULES §5–§8 to fill gaps (quest XP, training, gear tiers) — **discover** options from wiki data, do not assume fixed paths
5. Output a phased plan with citations

See [`examples/`](examples/) for more prompt patterns.

## Regenerating data

```bash
cd indexer
pip install -r requirements.txt
python3 build.py --with-graph          # initial full build
python3 wiki.py update                 # later: sync + rebuild + finalize
```

To patch specific pages without a full rebuild:

```bash
python3 wiki.py fetch -t "Archaeology" --rebuild --finalize
python3 wiki.py remove -t "Unwanted page" --rebuild --finalize
```

`MANIFEST.json` is rewritten on finalize with your local build stats — it is gitignored (see `MANIFEST.example.json` for the schema).
